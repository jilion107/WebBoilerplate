# WebBoilerplate

技术栈
-UI库：React & React-Dom  

-UI组件：Antd  

-路由：React-Router & History  

-框架：Alt  

-JS：ES6  

-样式：Less  

-通信：Ajax  

-打包构建：Webpack  

-包管理：Npm  

    
    
  
快速开始  

git clone https://github.com/jilion107/WebBoilerplate.git  

cd bo/src  

npm install  

npm run build  

npm start  

open http://localhost:8080/login
